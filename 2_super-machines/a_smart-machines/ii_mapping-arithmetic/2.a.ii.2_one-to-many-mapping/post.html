Objective:

Use several "Arrays" to store several different transfer functions; use these arrays to map an incoming stream of numbers to several different destinations.

Steps and observations

  1. Draw four different transfer functions in the four different arrays
  2. Move the "input" slider and analyze the results in the "output" sliders
  3. Change the transfer functions and repeat step 2

Comments and Challenges

  • Use this patch to map one sensor to several different servo rotations so as to create a coordinated cooperative action among all servos
  • Use this patch to create a sound composition with four independent tracks
