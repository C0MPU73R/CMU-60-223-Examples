Introduction to Physical Computing

This github repository 'physcomp-examples' provides the course materials for the Introduction to Physical Computing course at Carnegie Mellon University. This course is being offered for the first time in Fall 2014 as part of the new IDeATe program. For details, see the IDeATe web site and the main course site.

Main Units

The course is divided into three main units. Each of following numbered folders contain demonstration programs and projects corresponding to a unit. These are intended to explain the ideas behind each assignment and provide sample code.

  • 1_energy-information-transduction
  • 2_super-machines
  • 3_super-powers
