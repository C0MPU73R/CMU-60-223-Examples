Exercise: Reading a Digital Sensor with Raspberry Pi GPIO

Objective: Read from a button attached to a pin on the Pi

Refer to the schematic to see how to build the circuit for this exercise.

Steps and observations

  1. Refer to the schematic to see how to build the circuit for this exercise.
  2. Set WiringPi pin 0 (GPIO-BCM 17) as an input:
    3. > gpio mode 0 input
  3. Read the state of pin 0. This should change with the state of the button:
    4. > gpio read 0
