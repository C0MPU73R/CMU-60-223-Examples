Exercise: Controlling a Lightbulb with PWM

Objective: Control a DC Lightbulb Using a PWM Pin on the Raspberry Pi

Importantly, WiringPi uses a different pin numbering scheme from the Pi Clobber. Enter 'gpio readall' to see a table that shows which BCM-GPIO pin numbers correspond to which WiringPi pins.

Steps and observations

  1. Control a lightbulb with PWM from pin 18. The Pi doesn't provide enough voltage for the lightbulb, so you will need to use an external power source with a 2N 3904 transistor. Connect the gate pin of the transistor to pin 1 from the Pi.
  2. Set WiringPi pin 1 (BCM-GPIO pin 18) to PWM mode. Note that this is the only pin on the Pi that supports hardware PWM, hence it is the only PWM pin that can be used with the gpio command:
    3. > gpio mode 1 pwm
  3. Set the lightbulb to half-brightness:
    4. > gpio pwm 1 512
